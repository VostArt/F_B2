# Frontend & Backend Development Practice

## 🛠 Стек технологий
- **Backend:** Node.js, Express, Swagger (OpenAPI)
- **Frontend:** React, Vite, Sass, Axios
- **Tools:** Postman, Git

## 📂 Структура проекта
- `practice_01` - Верстка (Sass/SCSS)
- `practice_02` - Основы Express (API)
- `practice_03` - Работа с API и Postman
- `practice_04` - Итоговое приложение (Fullstack интернет-магазин)

## 🚀 Как запустить Итоговый проект (Практика 4-5)

Проект состоит из двух частей: Сервер и Клиент. Их нужно запускать в разных терминалах.

### 1. Запуск Сервера (Backend + Swagger)
```bash
cd practice_04/server
npm install
node server.js
```
### 2. Запуск Клиента (Frontend)
```bash
cd practice_04/client
npm install
npm run dev
